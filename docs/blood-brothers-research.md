# Blood Brothers (DeNA, 2012) — Ресёрч механик

## Откуда взялась механика

Прямая родословная жанра **social card battler RPG**:
- **2010** — Dragon Collection (Konami/Mobage) — нулевой пациент карточных мобильных RPG в Японии
- **2011** — Rage of Bahamut (Cygames/DeNA) — завоевал #1 в Google Play США, чисто карточный баттлер с синтезом/эволюцией
- **2012** — Blood Brothers (DeNA) — добавила RPG-надстройку: варлорд-аватар, захват врагов в бою, тёмный фэнтезийный нарратив, сместила RoB с первого места

---

## Ключевые механики

### Варлорд (Warlord)
- Аватар игрока, всегда присутствует в отряде
- Расы: Knight, Samurai, High Elf, Dark Elf, Ape, Lizard-man, Dwarf, Goblin — разные стартовые статы
- Растёт в уровнях, прокачивает ATK / DEF / HP
- На уровне 50+ может «поглотить» Epic/Legendary фамильяра (уровень 99) — наследует его навык, фамильяр уничтожается

### Фамильяры (Familiars)
Статы: **HP, ATK, DEF, AGI** (инициатива), **WIS** (магия)

| Редкость | Обозначение | Макс. уровень |
|----------|-------------|---------------|
| Common | C | 30 |
| Uncommon | UC | 40 |
| Rare | R | 70 |
| Epic | EP | 99 |
| Legendary | L | 99 |
| Mythic | M | 99 |

### Бой (Combat)
- Пошаговый, **автоматический** — игрок не управляет каждым ходом
- Порядок хода определяется **AGI**
- **Pre-Combat навык** — срабатывает 1 раз перед боем (бафф/дебафф)
- **Combat навык** — активируется 1 раз за ход (атака, хил, воскрешение)
- Вся логика на сервере

### Захват и пакты (Capture / Pact)
Два способа получить фамильяра:

**1. Захват в бою:**
- Побеждённый враг может стать пленником (Captive)
- Blood Meal — шанс < 100% на создание пакта
- Heartsblood — редкий, гарантирует 100% успех

**2. Призыв (Gacha):**
- Copper Pact → Silver → Scarlet → Blessed Pact (лучший)
- Платная валюта (Mobacoin) = доступ к лучшим пулам
- Стандартная гача-механика с рандомным результатом

### Эволюция (Evolution)
- Два идентичных фамильяра одной линии сливаются
- Результат: тот же тип, но статы ×1.1–1.2, звёздный рейтинг растёт
- 1★ + 1★ → 2★; 2★ + 2★ → 3★ и т.д.
- Стоит золото (игровая валюта)

### Ивенты
- 3–4 ивента в месяц, каждый ~1 неделю
- Типы: PvP (самые популярные), кооп-рейды, прокачка
- По данным GDC 2013 (презентация DeNA) — во время ивентов **ARPU удваивался**
- Эксклюзивные фамильяры только через ивенты

## Открытый код и документация

**Исходного кода нет.** Вся серверная логика закрыта, серверы отключены ~2014–2015.

Что публично сохранилось:
- **Fandom-вики** — `bloodbrothersgame.fandom.com` — механики, статы, эволюция, навыки варлорда
- **GDC Vault 2013** — `gdcvault.com/play/1018073` — официальная презентация DeNA об ивентной системе
- APK-файлы клиента на Uptodown (но логика была на сервере)

## Что нужно для воссоздания со своим дизайном

### Фронтенд
- Unity (мобиль) или Phaser/PixiJS (браузер) для 2D карточного RPG
- Карточный UI (иллюстрации + статы), анимации боя, меню коллекции

### Бэкенд (вся критичная логика — на сервере)

| Компонент | Назначение |
|-----------|-----------|
| Auth | Аккаунты (email / Google / Apple) |
| Player store | Коллекция фамильяров, валюта, прогресс |
| **Gacha engine** | Взвешенные вероятности по редкости + pity-счётчик (гарант) |
| **Battle resolver** | Порядок ходов по AGI, применение навыков, проверка HP |
| Dungeon system | Зоны, расход энергии, дроп Captives |
| Evolution calculator | Правила слияния, расчёт бонуса статов |
| Event engine | Тайминг ивентов, очки, ранкинг, таблицы наград |
| Currency / shop | Обработка покупок, защита от читерства |

### База данных
- **Redis** — горячие данные (сессии, текущий бой, энергия)
- **PostgreSQL** — постоянные данные (профиль, коллекция, история транзакций)
- **JSON-конфиги** — контент (таблицы фамильяров, навыков, данжей)

### Ключевые алгоритмы

**Порядок хода в бою:**
```
сортировка отряда по AGI
→ применение Pre-Combat буффов (1 навык)
→ поочерёдные атаки/навыки
→ проверка HP после каждого хода
→ победа / поражение
```

**Gacha roll:**
```
взвешенный случайный выбор по редкости
+ pity-счётчик (гарантированный редкий через N попыток)
генерируется на сервере, клиент только отображает результат
```

**Эволюция:**
```
выбор двух фамильяров → проверка совместимости линии
→ новый фамильяр с базовыми статами × 1.1–1.2
→ звёздный рейтинг +1
```

### Контент (основной труд)
- Таблицы фамильяров: ID, редкость, базовые статы, кривая роста, эволюционная линия, навык
- Таблицы навыков: тип, эффект, формула урона/хила, условие активации
- Таблицы данжей: зоны, волны врагов, дроп-таблицы
- Гача-пулы: наборы фамильяров по тиру пакта с весами редкости

## Вывод

Механика Blood Brothers **полностью воспроизводима** — вся игровая логика задокументирована в вики и GDC-материалах. Уникального алгоритма нет: взвешенный гача + пошаговый автобой с сортировкой по AGI + эволюция через слияние. Главная трудность — **объём контента** (сотни фамильяров с балансом) и **серверная инфраструктура** для честного гача без читерства.
