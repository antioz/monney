# Поляна-2 — Design Spec (draft)

> Статус: в процессе разработки дизайна

---

## Концепция

**Название:** Поляна-2  
**Жанр:** Гача-RPG (по механике Blood Brothers) с сеттингом оригинальной Поляны  
**Платформа:** Telegram Mini App (первый клиент), бэкенд платформонезависимый  
**Цель:** Демо → MMORPG

---

## Сеттинг

Берём сеттинг оригинальной Поляны: футбольные хулиганы, фракции (КБ, КС, БГ), банды, городские районы. Вместо autochess — механика Blood Brothers: варлорд + коллекция бойцов + пошаговый автобой.

---

## Ключевые решения

| Вопрос | Решение |
|--------|---------|
| Платформа | Telegram Mini App (HTML/CSS/JS) |
| Авторизация | Telegram initData — без регистрации |
| Бэкенд | Node.js + Express, REST API |
| Варлорд | «Старший» — именной аватар игрока |
| Получение бойцов | Захват в бою + вербовка на рынке |
| Режим демо | Одиночная кампания по районам |
| Расширение | Тот же бэкенд → MMORPG (PvP + кооп-рейды) |

---

## Архитектура

```
[Telegram Mini App]
        ↕ REST API (JSON)
[Node.js + Express — бэкенд]
        ↕
[PostgreSQL]   [Redis]   [JSON-конфиги]
Профили,       Сессии,   Бойцы, навыки,
бойцы,         бои       районы
прогресс
```

### Серверные модули

**Демо:**
- `Auth` — верификация Telegram initData
- `PlayerService` — профиль Старшего, коллекция бойцов, энергия
- `BattleResolver` — пошаговый бой: порядок по AGI, Pre-Combat навык, Combat навыки
- `CaptureService` — захват бойцов после боя + вербовка на рынке
- `CampaignService` — районы города, волны врагов, дроп, прогресс

**MMORPG (не в демо, архитектурный задел):**
- `WorldService` — карта города, захват районов фирмами
- `PvPService` — забивы фирма vs фирма
- `WebSocket` — реалтайм для PvP и кооп-рейдов

## Игровые механики

### Старший (варлорд)
- Аватар игрока, всегда в отряде
- Характеристики: HP, ATK, DEF
- Фракция задаётся при создании персонажа (КБ / КС / БГ)
- При высоком уровне может поглотить навык топового бойца

### Бойцы (фамильяры Поляны)
Статы: **HP, ATK, DEF, AGI** (порядок хода), **STR** (стойкость/воля)

Редкость адаптирована под сеттинг:

| Редкость | Название | Макс. уровень |
|----------|----------|---------------|
| Common | Залётный | 30 |
| Uncommon | Свой | 40 |
| Rare | Проверенный | 70 |
| Epic | Авторитет | 99 |
| Legendary | Легенда района | 99 |

Архетипы из оригинальной Поляны: Подписной спортик, Отморозок, Выездной, Молодой, Хардкор.

### Бой
- Пошаговый, автоматический
- Порядок хода по AGI
- Pre-Combat навык — 1 раз перед боем
- Combat навык — 1 раз за ход
- Вся логика на сервере, клиент только отображает результат

### Захват и вербовка
- **Захват в бою:** побеждённый боец может сдаться → предложить «уважение» (расходник, шанс < 100%) или «базар» (гарант, редкий)
- **Вербовка:** рынок района — конкретный боец за монеты

### Кампания (демо)
- Карта районов города (3–5 районов в демо)
- Каждый район = серия боёв с нарастающей сложностью
- Финальный бой района = мини-босс
- Расход энергии на бои, восстанавливается со временем

### Расширение до MMORPG
- Те же районы становятся зонами общего мира
- Фракции = гильдии, захват района = PvP-событие
- Кооп-рейды против крупных банд/боссов
- WebSocket для реалтайм PvP, REST остаётся для всего остального

|-----------|-----------|
