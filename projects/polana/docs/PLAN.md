## Сеттинг

Футбольные хулиганы дерутся на поляне в городском парке. Жанр — «забив» (организованная групповая драка по правилам фанатской субкультуры).

**Поле боя:** Поляна в городском парке, трава, деревья по краям.

**Фоновые персонажи (не играбельные, создают атмосферу):**
- Дед
- Мама с коляской
- Велосипедист
- 2 человека с разных сторон с камерой (обязательно!)
- «Тренеры» и «смотрящие» по краям

**Визуальный стиль:** Реалистичный, референсы — реальные видео забивов. Парни в одинаковых футболках, спортивных штанах, разные по характеру и характеристикам.

**Референсы:** Предоставлены пользователем (фото с Facebook/TikTok с тегами #оф #забив #offnik).

## Игровая механика

### Уровни системы (Dota Underlords-style)

**Уровень 1 — Фракция (выбирается до матча):**
Игрок выбирает «фирму» — фанатскую группировку клуба. Даёт пассивные бонусы всей команде.

**Уровень 2 — Архетип (тип юнита):**

| Архетип | Механика |
|---|---|
| Подписной спортик | Контроль + стабильный качественный дамаг |
| Отморозок | АоЕ урон, может бить своих (хаотичный) |
| Выездной | Бафф союзников, опыт и поддержка |
| Молодой | Высокая дисперсия на СПОСОБНОСТИ: X% шанс сработать с 3× эффектом, X% шанс полностью провалиться. Применяется к спецприёму — так дисперсия заметна и драматична |
| Хардкор | Дорогой, стабильный, основа команды. Высокий HP, умеренный константный дамаг, аура-бафф для ближайших союзников |

**Уровень 3 — Класс (конкретный юнит внутри архетипа):**

| Архетип | Классы |
|---|---|
| Подписной спортик | МСМК по боксу, Чемпион Беллатура, Двоюродный брат из спецназа, Знакомый боксерчик |
| Отморозок | Сидевший, Бешеный торч, Детдомовец, БС |
| Выездной | Боевой ультрас, Пьянь |
| Молодой | TBD |
| Хардкор | TBD |

*(Список неполный — расширяется по мере разработки)*

### Слияние юнитов (must-have для демо)

Как в Underlords/TFT: 3 юнита 1★ → 1 юнит 2★ → 3 юнита 2★ → 1 юнит 3★.  
В ПОЛЯНЕ вместо звёздочек — **принадлежность к банде** (badge + название меняется при каждом слиянии).

## Визуальный стиль

**Референс:** Totally Accurate Battle Simulator (TABS) — low-poly персонажи с ragdoll физикой, намеренно «деревянная» анимация. Не баг, а фича: не нужны дорогие анимации, процедурная физика делает бой живым.

**Дифференциация юнитов через детали модели:**

| Элемент | Что кодирует |
|---|---|
| Форма лица (квадратный/узкий/широкий) | Архетип |
| Шрамы, татухи | Класс юнита |
| Нашивка Stone Island / Burberry / Fred Perry | Фракция |
| Полоски на Adidas / Nike swoosh | Банда / уровень звезды |
| Цвет футболки | Фракционный цвет (КБ — красно-белые и т.д.) |

**Принцип:** детали читаются даже при маленьком размере юнита на сетке — они работают как визуальные теги, а не украшение.

## Технический стек

- **Движок:** Unity 6 (6000.0 LTS)
- **Язык:** C#
- **Опенсорс основа:** форк **ChraumaTactics** (`github.com/TaillepierreN/ChraumaTactics`)
  - Активен (авг 2025), есть: расстановка на сетку, бой, синергии, 16+ юнитов, мультиплеер
  - Нужно дописать: слияние 3→1 (отдельный менеджер)
- **MCP интеграция:** Unity MCP — варианты:
  - [Unity официальный MCP](https://docs.unity3d.com/Packages/com.unity.ai.assistant@2.0/manual/unity-mcp-overview.html)
  - [CoplayDev/unity-mcp](https://github.com/CoplayDev/unity-mcp)
  - [CoderGamester/mcp-unity](https://github.com/CoderGamester/mcp-unity)
- **Разработка:** Claude Code + Cursor, агенты управляют Unity через MCP
- **Деплой:** PC (первый), Steam (потенциально)

**Подход:** Форк ChraumaTactics + замена визуала и данных юнитов на дизайн ПОЛЯНЫ. Spec разбит по зонам — каждый блок берёт отдельный агент.
